## üéØ Overview

XHost is a developer-first SaaS platform inspired by Vercel. It provides Git-based deployments, automated builds, serverless functions, and a foundation for AI-assisted developer workflows. Built with modern technologies and designed for scale.

**Live Demo:** [https://xhost.live](https://xhost.live)
